# 3D 모델링 (안경 프레임)

Fusion 360으로 설계한 안경 프레임 하우징입니다. 캡스톤 디자인 원본 폴더의 "3D 모델링"
폴더에서 가져왔습니다 (파일명만 영문으로 정리했고, 내용은 원본 그대로입니다).

- `cooking_assist_glasses.f3d` — Fusion 360 네이티브 파일입니다 (편집 가능).
- `cooking_assist_glasses.step` — STEP 포맷입니다 (다른 CAD 툴에서도 열립니다).
- `cooking_assist_glasses.stl` — 3D 프린팅용 메시입니다 (실제로 이 파일로 출력한 사진은
  [../photos/hardware-3d-print-1.jpg](../photos/hardware-3d-print-1.jpg),
  [../photos/hardware-3d-print-2.jpg](../photos/hardware-3d-print-2.jpg) 참고).
- `render-isometric.png` — 조립된 형태의 아이소메트릭 렌더입니다 (도면 양식).
- `render-exploded-view.png` — 분해도(exploded view)입니다. 위쪽부터 안경 프레임 상단부,
  카메라/센서가 들어가는 중단부, 하단 커버 순서로 3파트 구조임을 보여줍니다.
